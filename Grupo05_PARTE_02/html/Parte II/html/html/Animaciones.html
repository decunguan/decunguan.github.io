ANIMACIONES

Aprende a crear animaciones HTML usando JavaScript

Introduccion

Para la creación de una animación empezamos con el boceto de una página web básica, creando un contenedor y un estilo propio


Ejemplo

<!DOCTYPE html>
<html>
<body>
<h1>Mi primera animación en Javascript/h1>
<div id="animation">La animación va aquí/div>
</body>
</html>
<script>
function eje1(){
document.getElementById("prueba1").innerHTML = "Aquí se presentará la animación";
}
</script>

Estilo de Animación

  • El elemento contenedor debe crearse con estilo = " position: relative".
  • El elemento de animación debe crearse con estilo = " position: absolute".


Ejemplo

<style>
#containerP1 {
width: 400px;
height: 400px;
position: relative;
background: skyblue;
}
#animateP1 {
width: 50px;
height: 50px;
position: absolute;
background: yellow;
}
</style>
</html>

Animacion Completa

Te presentamos el código de la aminacion de js


Ejemplo

<script>
function myMove() {
let id = null;
const elem = document.getElementById("animate");
let pos = 0;
clearInterval(id);
id = setInterval(frame, 5);
function frame() {
if (pos == 350) {
clearInterval(id);
} else {
pos++;
elem.style.top = pos + "px";
elem.style.left = pos + "px";
}
}
}
</script>
