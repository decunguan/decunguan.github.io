Lista de Nodos

Un objeto NodeList es una lista (colección) de nodos extraídos de un documento.

Introduccion

Algunos navegadores (más antiguos) devuelven un objeto NodeList en lugar de HTMLCollection para métodos como getElementsByClassName().

Todos los navegadores devuelven un objeto NodeList para la propiedad childNodes.

La mayoría de los navegadores devuelven un objeto NodeList para el método querySelectorAll().


Ejemplo

El siguiente código selecciona todos los <p> nodos en un documento


<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<p>Hola Mundo!</p>
<p>Hola Pepe!</p>
<p id="demo"></p>
<script>
const myNodelist = document.querySelectorAll("p"); document.getElementById("demo").innerHTML = "El innerHTML del segundo parrafo es : " + myNodelist[1].innerHTML;
</script>
</body>
</html>

COLECCION LONGITUD

La propiedad length define el número de nodos en una lista de nodos

La propiedad length es útil cuando desea recorrer los nodos en una lista de nodos

Ejemplo 2

Cambia el color del texto de todos los elementos <p> en una lista de nodos:


<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<p>Hola Mundo!</p>
<p>Hola Pepe!</p>
<p id="demo"></p>
<p>Click al boton para cambiar el color de todos los elementos p.</p>
<button onclick="myFunction()">Probar</button> <script>
function myFunction() {
     const myNodelist = document.querySelectorAll("p");
     for (let i = 0; i < myNodelist.length; i++) {
          myNodelist[i].style.color = "red";
     }
}
</script> </body>
</html>
